List of tags

Tags in this list can be used to filter for screen time of the associated face identities using the {{ parameters.onscreen_face }}="tag: ..." predicate.

Tags (all faces)

The following tags are available on all faces, regardless of whether we have an identity label.

Name Meaning
male A face identified as male by our gender classifier.
female A face identified as female by our gender classifier.
host A face identified as a host, based off of our list of known hosts.
nonhost A face identified that is not identified as a host.

Tags (derived from identity)

The following tags are obtained from DBpedia using the names of faces that we identified using Amazon Rekognition's celebrity detection API. Filters on tags from this list will only apply to faces for which we have an associated identity (44% of the faces). For example, the filter {{ parameters.onscreen_face }}="tag: journalist" incudes the screen time of all faces whose identiies have the journalist tag.

The source column of the table below shows how tags were obtained. Tags with "dbpedia" as the source are parsed from DBpedia properties. Tags with "derived" are derived from DBpedia properties based on a small number of heuristic parsing rules; for example, a person with the tag "english_people_of_french_descent", will be given the derived tags "english" and "people_of_french_descent". There may be multiple rows for a single tag in the table below if the tag is both an original property from DBpedia and a derived one. In this circumstance, searching in the tool considers the union of the derived and dbpedia tags.
