List of person-attributes

This page contains a list of attributes obtained from dbpedia using names of faces identified using AWS Rekognition Celebrity detection.

Attributes in this list can be used to filter for screen time pertaining to faces using the onscreen.face1="attr: ..." predicate. Note that this filter will only apply to faces for which we have an associated identity. The result is a union of all people who have the tag in question.
