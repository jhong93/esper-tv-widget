List of person-tags

This page contains a list of tags obtained from DBpedia using the names of faces that we identified using Amazon Rekognition's celebrity detection API.

Tags in this list can be used to filter for screen time of the associated face identities using the onscreen.face1="tag: ..." predicate. Note that this filter will only apply to faces for which we have an associated identity (44% of the faces). For example, the filter onscreen.face1="tag: journalist" incudes the screen time of all faces whose identiies have the journalist tag.
